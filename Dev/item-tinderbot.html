TinderBot

Details

Done with: Python
Date: 10 March, 2020
Category: Robot
Downloadable: Yes

The tinderbot is a handytool for those who are too lazy to swipe everyday. Let the robot match people for you, but be carefull, you may have bad surprises... The robot can be executed for as long as you wish, just type in the minutes you want the execution to last. It also lets you choose the location where you want to swipe, and a maximum height which only works if the person has indicated it in its description. Finally, you can add good and bad words, that will make the robot swipe right (good words) or left (bad words) if the word appears in the description. Use it at your own risks!

Help

Installation:
Once the file has been downloaded, unzip it and run the file named: "tinderbot.exe". The first time it will ask you to log in to your tinder account, then it will make it automatically.
